# 📖 About Repository

Welcome to the **CSA0609 – Design and Analysis of Algorithms Laboratory Repository**.

This repository contains all Python laboratory programs implemented as part of the **Design and Analysis of Algorithms** course offered at **Saveetha Engineering College**.

The programs cover almost every important algorithmic paradigm taught in the course including

- Divide and Conquer
- Brute Force
- Dynamic Programming
- Greedy Algorithms
- Backtracking
- Searching Algorithms
- Sorting Algorithms
- Matrix Algorithms
- Graph Algorithms
- Optimization Problems
